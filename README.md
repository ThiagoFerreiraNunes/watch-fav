# watch-fav

WatchFav é uma aplicação web full stack que permite aos usuários favoritar seus filmes e séries preferidos para assistir mais tarde.
Cada usuário pode explorar os títulos disponíveis, marcar como favoritos e gerenciar sua lista pessoal de forma simples e intuitiva.

O objetivo do WatchFav é oferecer uma maneira prática e organizada de acompanhar o que o usuário deseja assistir novamente ou descobrir no futuro.

Desenvolvido com React no frontend e Spring Boot no backend, o WatchFav oferece uma experiência moderna, responsiva e confiável para o gerenciamento de preferências de entretenimento.
